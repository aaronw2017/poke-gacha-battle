# Poké-Gacha Battle

A browser-based Pokémon gacha + battle game (single HTML file).

## Play
Open `index.html` in a browser, or visit the GitHub Pages site once published.

## Features
- Gacha spinner with rarity tiers (common → mythic)
- Turn-based battles with type chart, custom movesets, and effects (counter, shield, lifesteal, double-strike, pierce)
- Pokédex collection, team builder, shop, and secret codes
- Multiple save accounts with file backup/restore
